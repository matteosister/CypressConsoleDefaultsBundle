### Installation

```json
{
    "require" {
        "cypresslab/console-defaults-bundle": "~0.3"
    }
}
```

register the bundle in your AppKernel.php

```php
$bundles[] = new Cypress\ConsoleDefaultsBundle\CypressConsoleDefaultsBundle();
```
